# 🌐 Sample Website

This project is a simple **HTML website structure** that demonstrates the use of a **header**, **navigation bar**, **content sections**, and **footer**.  
It’s designed for beginners learning web development basics such as **layout organization**, **semantic HTML tags**, and **external CSS linking**.

---

## 🧩 Project Structure

```

sample-website/
│
├── index.html        # Main HTML file
├── style.css         # External CSS for styling
└── README.md         # Project documentation

````

---

## 📄 Description

The website consists of:

- **Header:** Contains a navigation bar with links such as *Home*, *About*, *Contact Us*, and *Address*.  
- **Main Content:** Includes multiple sections (boxes and content areas) with placeholder text (`Lorem ipsum`) for structural understanding.  
- **Footer:** Provides extra navigation links like *Education*, *Skills*, *Projects*, and *Interview*.  
- **Image Section:** Displays a sample vector image at the bottom of the page for design demonstration.

---

## 💅 Features

- Clean and simple **HTML structure**  
- Organized layout using `<header>`, `<nav>`, `<div>`, and `<footer>` tags  
- Linked external stylesheet (`style.css`) for design customization  
- Responsive setup using `<meta name="viewport">` for mobile compatibility  
- Embedded example image using an external URL  

---

## 🖋️ How to Use

1. **Download or clone** the repository:
   ```bash
   git clone https://github.com/yourusername/sample-website.git
````

2. Open the project folder.
3. Double-click on `index.html` to open it in your browser.
4. Customize the design by editing `style.css`.
